## 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes e-commerce sales data using Power BI to generate insights on sales performance and profit trends.

## 🛠 Tools Used
- Power BI
- Excel

## 🔧 Data Cleaning
Data cleaning and transformation were performed within Power BI using Power Query:
- Removed null values
- Changed data types
- Filtered incorrect records

## 📊 Dashboard Features
- Sales by Category
- Profit Analysis
- Monthly Trends
- KPI Cards
